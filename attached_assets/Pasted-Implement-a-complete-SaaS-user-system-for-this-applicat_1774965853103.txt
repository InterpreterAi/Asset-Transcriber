1. USER AUTHENTICATION

Add a secure authentication system with:

• Sign Up  
• Login  
• Logout  
• Forgot Password  
• Reset Password  
• Change Password  
• Google Sign-In (OAuth)

Passwords must be stored using bcrypt hashing.

Email verification is required for new email-based accounts.

Google login must allow users to sign in using their Google account without creating a password.

When a user signs in with Google for the first time, automatically create a user record and start the free trial.

Each user record must include:

user_id  
email  
password_hash (if email signup)  
google_account_id (if Google signup)  
plan_type  
trial_start_date  
trial_end_date  
minutes_used_today  
minutes_used_this_month  
last_daily_reset  
last_monthly_reset  
active_session  
signup_ip  
device_fingerprint

2. FREE TRIAL SYSTEM

Every new user automatically receives a trial:

Trial duration: 14 days  
Daily usage limit: 5 hours (300 minutes)

Logic:

• Trial starts when account is created
• Trial expires after 14 days
• Usage resets every 24 hours

If daily limit is reached:

Display message:

"Daily trial limit reached. Please upgrade to continue."

If trial expires:

Display message:

"Trial expired. Please subscribe to continue using the service."
